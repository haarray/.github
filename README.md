# Haarray GitHub Organization Profile Repo

This folder is prepared for the GitHub organization profile repository.

GitHub expects the repository name to be:

```text
.github
```

Inside that repository, GitHub renders:

```text
profile/README.md
```

Recommended organization repo structure:

```text
.github/
└── profile/
    ├── README.md
    └── assets/
        ├── haarray-logo-horizontal-transparent.png
        ├── haarray-app-icon-512.png
        └── haarray-github-avatar.png
```

This is safe to push publicly. Do not push `.haarray-local/`.
